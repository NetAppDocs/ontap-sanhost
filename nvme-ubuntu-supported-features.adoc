= Learn about ONTAP support and features for Ubuntu hosts
:hardbreaks:
:toclevels: 1
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ./media/

[.lead]

ONTAP supports the following SAN host features regardless of the ONTAP version running on your system setup.

[cols="75,25",options="header"]
|===

|Feature | Ubuntu host version
|The NetApp plug-in in the native package displays ONTAP details for NVMe/FC namespaces.
a|22.04 or later
|NVMe and SCSI traffic is supported on the same host on a given host bus adapter (HBA), without the explicit dm-multipath settings to prevent claiming NVMe namespaces
a|22.04 or later
|In-kernel NVMe multipath is enabled by default for NVMe namespaces. Therefore, no explicit configuration is required
a|22.04 or later
|===

NOTE: For details on supported configurations, see the link:https://mysupport.netapp.com/matrix/[Interoperability Matrix Tool^].
