= Learn about ONTAP support and features for Oracle Linux hosts
:hardbreaks:
:toclevels: 1
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ./media/

[.lead]
Features supported for host configuration with NVMe over Fabrics (NVMe-oF) varies based on your version of ONTAP and RHEL.

[cols="50,25,25",options="header"]
|===

|Feature | RHEL host version| ONTAP version

|Secure in-band authentication is supported over NVMe/TCP between a RHEL host and an ONTAP controller
a|9.4 or later
 |9.12.1 or later
|NVMe/TCP provides namespaces using the native `nvme-cli` package
a|8.2 or later (TBC)
 |9.10.1 or later  
|NVMe/TCP is a fully supported enterprise feature
a|9.0 or later  (TBC)
 |9.10.1 or later
|NVMe and SCSI traffic is supported on the same host using NVMe multipath for NVMe-oF namespaces and dm-multipath for SCSI LUNs
a|8.2 or later  (TBC)
 |9.4 or later
|===

include::_include/nvme/nvme-cli-note.adoc[]
