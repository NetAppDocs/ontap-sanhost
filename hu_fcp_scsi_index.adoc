= Learn about configuring ONTAP SAN hosts with FCP and iSCSI
:hardbreaks:
:toclevels: 1
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ./media/

[.lead]
You can configure certain SAN hosts for FCP or iSCSI with ONTAP as the target. If your host operating system (OS) is supported by SAN Host Utilities software, you first install the relevant OS host utility package, which includes the SAN tool kit. Then, for all host OS types, you configure the multipath settings for ONTAP LUNs.
