Enable FC/NVMe on the Windows initiator host.

.Steps
. Install Emulex HBA Manager utility on the Windows host.

. On each of the HBA initiator ports, set the following HBA driver parameters:
+
* EnableNVMe = 1
* NVMEMode = 0
+
. Reboot the host.
